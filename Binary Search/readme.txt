Bianry Search Readme

Calling syntax:
BinarySearch.py <-f> <filename> <-t> <target value>

The filename must be a valid path to a file containing a sorted list of integers.
The target value must be an integer, but does not neccesarily have to exist in the given list.

Error codes:
0 - No error. Success
1 - No arguments specifed
2 - Flag not matched to argument or argument not matched to flag
3 - Non-numeric target specified
4 - Invalid argument flag
5 - No file provided
6 - No target provided
